 The following files have been provided:

- LVA_no_XT30_V1
	- Formats: .stl, .stp, .ipt
	- Use: this is the launch vehicle adapter (LVA) that YOU are expected to manufacture and include in your designs. It is expected that this will be done with a 3d printer, please contact us at rocketry@ukseds.org if you do not have access to one of these. This version of the LVA has no cutout for the XT30 connector. As a reminder, as per requirement NRC MRD-CPD-004, you will be providing power to "the customer's" payload. You have two options to power this payload, either from the bottom or the top through the LVA. This LVA file is provided for if you want to provide power from the bottom. This allows you to place ejection charges behind the LVA and since there is not cutout for the XT30 connector, this should mitigate the hot gases from damaging the payload.
	- Can I modify this file? Yes - you may modify this file to help integrate it into your design. However, please remember that the mating interface is fixed as this needs to match "the customer's" payload. Therefore, we would recommend only modifying the parts' diameter and if you do so, you will need to provide some longer bolts to mount the payload. If you intend to modify this part, we recommend that you drop us a message at rocketry@ukseds.org with a .stl of your modified version and a description of what you have changed and we can check it quickly for any issues.

- LVA_XT30_V1
	- Formats: .stl, .stp, .ipt
	- Use: this is the other version of the launch vehicle adapter (LVA) that YOU are expected to manufacture and include in your designs. It is expected that this will be done with a 3d printer, please contact us at rocketry@ukseds.org if you do not have access to one of these. This version of the LVA has a cutout for the XT30 connector. As a reminder, as per requirement NRC MRD-CPD-004, you will be providing power to "the customer's" payload. You have two options to power this payload, either from the bottom or the top through the LVA. This LVA file is provided for if you want to provide power to the top of the payload, through the LVA.
	- Can I modify this file? Yes - you may modify this file to help integrate it into your design. However, please remember that the mating interface is fixed as this needs to match "the customer's" payload. Therefore, we would recommend only modifying the parts' diameter and if you do so, you will need to provide some longer bolts to mount the payload. If you intend to modify this part, we recommend that you drop us a message at rocketry@ukseds.org with a .stl of your modified version and a description of what you have changed and we can check it quickly for any issues. You may also modify the size of the XT30 cutout. We highly recommend carrying out a fit check to see if your XT30 connector passes through this as early as possible.
